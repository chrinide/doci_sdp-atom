1 v2DM-DOCI
2 =========
3 This program implements a v2DM solver for the DOCI N-representability constraints.
4 Both the boundary point method and the potential reduction method have been
5 implemented. Two orbital optimizer are available:
6 - A simulated annealing code (in the extern directory)
7 - Jacobi rotations based local minimalisatie
8 
9 Build
10 -----
11 To build this, you need a C++11 compiler (GCC 4.8 or newer, Clang 3.3 or newer),
12 the HDF5 libraries and the blas and lapack libraries. The Makefile is quite
13 simple, adjust the compilers and header/libraries as needed for your system.
14 
15 Input
16 -----
17 The program needs symmetry-adapted electron integrals from [PSI4](https://github.com/psi4/psi4public).
18 You can extract them using a plugin found in the extern/mointegrals.cpp file.
19 
20 Documentation
21 -------------
22 Everything is documented with doxygen. Run `make doc` to build the HTML docs.

26 Example
27 -------
28 In the directory `example`, all the input files for the He dimer (interatomic distance is 10 Bohr)
29 can be found:
30 - `He2-integrals.dat`: This is the input file for PSI4 to generate the one- and two-electron integrals
31 - `mo-ints-he2-cc-pvdz-010.00.h5`: the Hartree-Fock molecular orbitals
32 - `so-ints-he2-cc-pvdz-010.00.h5`: the integrals in the orthogonalized, symmetry-adapted basis
33 - `unitary-mo-he2-cc-pvdz-010.00.h5`: the orthogonal transformation from the orthogonalized,
34  symmetry-adapted basis to the Hartree-Fock molecular orbital.
35 
36 You can run the v2DM-DOCI optimisation with:
37 `../doci_bp -i so-ints-he2-cc-pvdz-010.00.h5 -u unitary-mo-he2-cc-pvdz-010.00.h -l`
38 
39 The will generate:
40 - `optimale-uni.h5`: the optimal basis transformation
41 - `optimal-ham.h5`: the one- and two-electron integrals in the optimal basis
42 - `optimal-rdm.h5`: the optimal 2DM
43 
44 In the file `output.txt`, the output of the optimisation is stored.
